Desktop Environments

Want a different look and user experience for your computer? Try out a new desktop environment!

GNOME Shell
GNOME

Get things done with ease, comfort and control.

Plasma
KDE

Simple by default, powerful when needed.

Cinnamon
Linux Mint Team

Cinnamon is a desktop environment which combines a traditional desktop layout with modern graphical effects.

MATE
Stefano Karapetsasi and Community

The MATE Desktop Environment is the continuation of GNOME 2. It provides an intuitive and attractive desktop environment using traditional metaphors for Linux and other Unix-like operating systems.

XFCE
XFCE Community

Xfce is a lightweight desktop environment for UNIX-like operating systems. It aims to be fast and low on system resources, while still being visually appealing and user friendly.


Programs

Many available programs on Linux are multi-platform, however some programs really stand out on the Linux platform.

GIMP Image Editor

GIMP is a cross-platform image editor available for GNU/Linux, OS X, Windows and more operating systems.

Kdenlive Video Editor

Kdenlive is an acronym for KDE Non-Linear Video Editor. It is primarily aimed at the GNU/Linux platform but also works on BSD and MacOS. It is currently being ported to Windows as a GSOC project.

Clementine Music Player

Clementine is a multiplatform music player. It is inspired by Amarok 1.4, focusing on a fast and easy-to-use interface for searching and playing your music.

Linux Kernel

About

The Linux kernel is the core to many operating systems. Linux is the most popular platform for servers and supercomputers, among other mediums. The kernel is entirely open-source and allows for code contributions from anybody. Learn more.

History

Created by Linus Torvalds, Linux was originally developed as a clone of MINIX with additional features and a legal license that did not limit it to only educational purposes. The source code was released on September 17th, 1991.
